#  Telecom X – Análisis de Evasión de Clientes 

##  Descripción del Proyecto

Este proyecto presenta un **análisis exploratorio de datos (EDA)** enfocado en la **evasión de clientes (Churn)** de la empresa ficticia **Telecom X**.

La compañía enfrenta una alta tasa de cancelaciones y requiere comprender los factores que influyen en la decisión de los clientes de abandonar el servicio. El análisis busca transformar los datos en información estratégica que sirva como base para **acciones de retención** y **futuros modelos predictivos**.

El trabajo se desarrolló utilizando **Python** y librerías de análisis de datos, aplicando conceptos de **ETL, limpieza de datos, visualización y análisis exploratorio**.

---

##  Objetivos

- Analizar el comportamiento de los clientes que cancelan el servicio.
- Identificar variables asociadas al churn.
- Explorar patrones en variables demográficas, contractuales y de consumo.
- Generar insights accionables para reducir la evasión de clientes.
